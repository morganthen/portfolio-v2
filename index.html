Remit

Invoicing web app

[2025]

Built with Next.js, TypeScript, Supabase (auth + PostgreSQL), shadcn/ui and Tailwind. Client and invoice management with validated forms powered by Zod and React Hook Form.

Each invoice carries a status — draft, sent, unpaid, paid, overdue and void, which conditionally renders an action menu, keeping the UI practical.

Shareable via a unique public URL, and a pre-filled mailto action. Chose void-over-delete after researching how professional invoicing tools handle financial records — invoices are immutable by design for better tax auditing trails.

Lullo

AI-narrated bedtime story generator

[2026]

Two AI APIs chained inside one user flow: Anthropic writes the story, ElevenLabs narrates it in natural voices. All behind OAuth on a multi-tenant Supabase model.

Full Stripe subscription lifecycle through signed webhooks. Subscribe, cancel, resume and period-end and a failed-payment grace state all keep the database and UI in sync. The billing layer is portable across future projects.

Live demo runs in Stripe test mode — walk the full subscription flow with card 4242 4242 4242 4242, any future expiry, any CVC. KYC is done; live mode is staged for launch.

Oculé

Speed-reading web app + Chrome Extension

[2026]

Rapid Serial Visual Presentation flashes text one word at a time at a fixed focal point, cutting saccadic eye movement out of reading so content goes faster.

Eased punctuation pauses keep the rhythm natural. Font size and WPM adjust mid-session, saved to localStorage. Fully client-side, no backend or accounts — the text is the user's own.

A companion Chrome extension, live on the Chrome Web Store with real installs, parses any article — point it at a Wikipedia page and it hands the body text to ocule.app.

Droptop

Bookmark manager

[2026]

Built twice on purpose. V1 was split-repo Node.js + Express + MongoDB with a separate Vite frontend and CORS. V2 rebuilds it in Next.js with PostgreSQL and Prisma — the long way first made API routes click as a layer over request/response.

Scrapes title, description and cover image via Open Graph on URL blur, the trick Pinterest uses on save. Prisma 7's breaking changes from v6 (connection URL moved to prisma.config.ts, new driver adapter) sent me to the docs over outdated tutorials.

Auth is hand-rolled JWT and bcrypt. Built in Express first, so the Next.js redo reinforced the model — named GET/POST/DELETE exports returning Response objects instead of res.send().
